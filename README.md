# 💰 FinPilot AI

FinPilot AI is an AI-powered personal finance platform designed for:

- 🎓 College and university students
- 🌍 International students living abroad
- 💼 Young professionals

## Features (Planned)

- User authentication
- Budget tracking
- Expense management
- Savings goals
- AI financial insights
- Spending analytics
- Responsive dashboard

## Tech Stack

- HTML
- CSS
- JavaScript
- Python (Flask)
- SQLite
- Git & GitHub

## Mission

To help young adults build healthy financial habits through AI-powered budgeting and financial education.
